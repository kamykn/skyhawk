# Install
## Plug
Installation with Plug

```
Plug 'kmszk/skyhawk'
```

## NeoBundle
Installation with NeoBundle

```
NeoBundle 'kmszk/skyhawk'
```

# Setting
Add to your .vimrc:

```
colorscheme skyhawk
```
